# OracleOne-BuscadorCEP
<p>Provavelmente, você já se deparou com algum formulário em que, ao digitar seu CEP, seu endereço foi retornado automaticamente, bastando apenas você complementar com o número. Essa aplicação em Java será um buscador de CEP.</p>
<p>Ela consumi a API Via CEP, recebe os dados e exibi o endereço na aplicação, grava os dados num JSON.</p>
<p>Exemplo inclui:</p>
<p>**Uma aplicação para consultar a API ViaCEP;</p>
<p>**Menu para o usuário informar o CEP para busca;</p>
<p>**Geração de um arquivo .JSON com os dados do endereço.</p>
